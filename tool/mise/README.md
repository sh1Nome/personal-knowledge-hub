# mise
miseは複数のプログラミング言語やツールのバージョンを管理できるツールマネージャー

## インストール
* グローバルにツールをインストール - `mise install -g`
    * ツールを全環境で利用可能にする
* ツールをインストール - `mise install`
    * `.mise.toml`や`.tool-versions`で定義されたツールをインストールする

## ツール管理
* インストール済みツールの一覧表示 - `mise ls`
    * 現在インストールされているツールとバージョンを表示する
* インストール可能なバージョンの一覧表示 - `mise ls-remote`
    * 指定したツールでインストール可能なすべてのバージョンを表示する
* ツール情報の表示 - `mise tool`
    * 利用可能なツールの情報を表示する
* ツールの検索 - `mise search`
    * 利用可能なツールを検索する
* ツールレジストリの表示 - `mise registry`
    * miseで利用可能なツールのレジストリを表示する

## タスク管理
* タスク一覧の表示 - `mise tasks`
    * 定義されているタスクの一覧を表示する
* タスクの実行 - `mise run`
    * 定義されたタスクを実行する

## その他のコマンド
* ツールのパスを表示 - `mise which`
    * 指定したコマンドの実際のパスを表示する
* コマンドの実行 - `mise exec --`
    * mise環境でコマンドを実行する
    * 短縮形: `mise x --`
* ツールのアップグレード - `mise upgrade`
    * インストール済みツールを最新バージョンにアップグレードする
    * 短縮形: `mise up`
