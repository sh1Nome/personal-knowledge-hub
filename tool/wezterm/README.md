# WezTerm

## コピーモード
* `<C-S-x>` - コピーモード
* `<C-S-f>` - 検索
    * `<C-n>` - 次の検索結果に移動
    * `<C-p>` - 前の検索結果に移動
    * `<C-r>` - パターンマッチングモードの変更
        * 大文字小文字を区別する
        * 大文字小文字を区別しない
        * 正規表現
    * `<C-u>` - 検索パターンをクリア
    * `<ESC>` - 検索オーバーレイをキャンセル
