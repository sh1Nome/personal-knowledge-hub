# Chrome
## アドレスバー検索
* `@ブックマーク pattern` - ブックマークを検索
* `@タブ pattern` - タブを検索
* `@履歴 pattern` - 履歴を検索

## DevTools
* Network → throttling - ネットワーク速度を制限できる

## 拡張機能
* Chrome Remote Desktop
* Google オフライン ドキュメント
* Google 翻訳
* Keepa - Amazon Price Tracker
* LINE
* Picture-in-Picture Extension (by Google)
* Sakura Check Linker
* Vue.js devtools
* YouTube™ 二重字幕
* ドライブ用アプリケーション ランチャー（Google）

## フォント設定を初期に戻す方法

1. Chromeをすべて終了する
1. chrome://version/ にアクセスする
1. 「プロフィール パス」に移動する
1. 配下の Preferences を開く
1. `webkit` を属性ごと削除する
1. Chromeを起動する
1. chrome://settings/fonts を確認する
