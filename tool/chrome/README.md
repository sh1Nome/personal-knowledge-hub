# Chrome
## アドレスバー検索
* `@ブックマーク pattern` - ブックマークを検索
* `@タブ pattern` - タブを検索
* `@履歴 pattern` - 履歴を検索

## DevTools
* Network → throttling - ネットワーク速度を制限できる
